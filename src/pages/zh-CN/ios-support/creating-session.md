# 创建题集

题集用于把一组试题组织成一次可重复练习的内容。你可以手动选择试题，也可以采纳 Practiso 的试题推荐，或将两种方式结合使用。

## 从最近练习开始作答

题集页会显示最近使用的练习卡片。每张卡片对应一次作答记录，颜色由作答记录自动分配，方便快速区分。

1. 打开 Practiso。
2. 进入题集页。
3. 点击最近练习卡片。
4. 进入作答页后，上下滑动切换题目。

如果你想先查看某个题集下的历史记录，可以先点击题集列表中的题集，再选择对应作答记录开始。

## 新建题集

1. 在题集页点击右上角的创建按钮。
2. 在新建题集面板中设置题集内容。
3. 选择要加入题集的试题，或使用推荐结果补充题目。
4. 确认后返回题集页，新的题集会出现在列表中。

## 手动选择试题

手动选择适合你已经知道练习范围的场景，例如复习某一次错题、某个章节或一组自定义题目。

1. 在新建题集面板中进入试题选择。
2. 勾选要加入题集的题目。
3. 返回新建题集面板确认选择结果。
4. 创建题集。

## 使用试题推荐

试题推荐适合你希望快速补齐练习内容，或想围绕已有题目扩展相关练习的场景。

1. 在新建题集面板中查看推荐试题。
2. 预览推荐内容，确认是否符合练习目标。
3. 采纳需要的推荐题目。
4. 创建题集，或继续手动添加其他题目。

## 开始作答

1. 在题集页点击要练习的题集或最近练习卡片。
2. 进入作答页后，通过上下滑动切换题目。
3. 如果设置了计时器，剩余时间会显示在屏幕底部。
4. 如果开启准确率显示，答错时会出现动画与振动反馈。

## 常见问题

### 为什么题集页有两种入口？

最近练习卡片用于快速继续或重新开始最近作答；题集列表用于先查看某个题集，再选择具体作答记录。

### 可以同时手动选题和使用推荐吗？

可以。你可以先手动选择核心题目，再采纳推荐题目补充练习范围。

### 计时器在哪里设置？

创建题集时选择同时开始练习，使用计时器轮盘选择需要的时长。也可进入现有题集详细页，在创建新练习时使用相同方式添加计时器。设置完成后，进入作答页，计时器会显示在屏幕底部。
