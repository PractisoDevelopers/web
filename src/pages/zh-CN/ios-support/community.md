# 参与社区

Practiso 社区用于公开分享题目和存档。你可以在社区页查询、预览、下载其他用户公开的存档，也可以上传自己的内容。

## 浏览社区存档

社区页会显示已公开的存档列表。每个存档都可以查看基本信息，并可进一步预览内容。

1. 打开社区页。
2. 浏览公开存档列表，或使用搜索框查找内容。
3. 点击存档进入预览页。
4. 查看存档中的题目和分类内容。

## 下载社区存档

1. 在社区页打开需要的存档。
2. 进入存档预览页。
3. 确认内容适合导入后，选择下载或导入。
4. 下载完成后，存档内容可用于本地题库或后续练习。

如果只想查看内容，可以停留在预览页，不必立即导入。

## 上传内容到社区

你可以从题目页或相关分享入口将内容上传到社区。

1. 选择要分享的题目或存档。
2. 打开分享菜单。
3. 选择上传到社区。
4. 按提示完成上传。

社区存档默认为公开内容。上传前请检查题干、选项、分类名称和其他信息，避免包含不适合公开的内容。

## 管理社区账户

点击社区页的用户按钮可查看当前账户信息。账户页支持以下操作：

- 修改用户名。
- 修改设备名。
- 退出登录。
- 停用账户。
- 显示凭证二维码。

用户名用于在社区中标识你的公开内容。设备名用于区分不同登录设备。

## 导出和导入登录凭证

显示凭证功能会把当前账户登录信息导出为二维码。你可以用 Practiso 的扫描功能导入，也可以通过其他应用识别链接后跳转导入。

1. 在社区页点击用户按钮。
2. 选择显示凭证。
3. 在另一台设备上扫描二维码，或识别二维码中的链接。
4. 按提示完成账户导入。

请妥善保管凭证二维码。拥有凭证的人可能可以登录同一个社区账户。

## 停用账户

如果不再使用社区账户，可以在账户页选择停用账户。停用前请确认你已经了解相关影响，例如无法继续用该账户上传或管理内容。

## 常见问题

### 社区存档是公开的吗？

是。上传到社区的存档会公开展示，其他用户可以查询、预览和下载。

### 可以只预览不下载吗？

可以。进入存档预览页后，你可以先查看内容，再决定是否下载或导入。

### 换设备后如何登录同一个账户？

在旧设备的账户页显示凭证二维码，然后在新设备上扫描或识别链接导入账户。
