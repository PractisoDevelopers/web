# 管理试题

Practiso提供题目页、聚类页、试题详细页和分类详细页，用于浏览、编辑、分类、分享和导入试题。

## 浏览题目

题目页列出题库中的所有题目。你可以从这里进入试题详情，也可以对题目执行预览、删除和分享操作。

1. 打开题目页。
2. 滚动浏览所有试题，或使用搜索框筛选内容。
3. 点击题目进入试题详细页。
4. 长按题目可预览内容。
5. 左滑题目可执行删除操作。

## 编辑试题

试题详细页支持浏览和编辑两种状态。

1. 在题目页点击需要修改的题目。
2. 在试题详细页点击编辑按钮。
3. 修改题干或选项内容。
4. 完成编辑后退出编辑状态。

你也可以在试题详细页重命名试题、分享试题，或查看该试题所属的分类。

## 分享试题到社区

题目页支持将题目分享到社区。

1. 在题目页选择要分享的题目。
2. 打开分享菜单。
3. 选择分享到社区。
4. 按提示完成上传。

上传到社区的内容会公开展示。分享前请确认题目内容可以公开。

## 浏览分类

聚类页列出所有分类。分类可帮助你按主题、章节或知识点整理试题。

1. 打开聚类页。
2. 点击分类进入分类详细页。
3. 长按分类可预览内容。
4. 左滑分类可执行删除操作。

## 管理分类内容

分类详细页会显示属于当前分类的试题。你可以查看题目、调整相关度，或将题目从分类中排除。

1. 进入聚类页。
2. 点击需要管理的分类。
3. 在分类详细页查看该分类下的试题。
4. 长按题目预览内容。
5. 打开题目操作菜单，修改相关度或从分类中排除。

## 使用语义搜索补充分类

分类详细页的套索按钮会根据分类名搜索相关试题，适合快速补齐某个主题下的题目。

1. 进入分类详细页。
2. 点击套索按钮。
3. 查看语义搜索结果。
4. 将相关题目添加到当前分类。

如果搜索结果不符合预期，可以调整分类名称后再试。

## 预览和导入存档

从系统文件管理器打开Practiso存档时，可以先查看内容，不需要立即导入数据库。

1. 在文件管理器中打开Practiso存档文件。
2. 进入存档预览页。
3. 上下滑动切换预览题目。
4. 确认内容后选择导入存档到题库。
5. 导入完成后返回题集页或继续浏览。

## 常见问题

### 删除题目会影响已有题集吗？

删除前请确认该题目不再需要。若题目已被用于题集或分类，删除后相关内容可能不再完整。

### 分类和题集有什么区别？

分类用于长期整理题库，题集用于组织一次或多次练习。你可以先用分类整理题目，再基于这些题目创建题集。

### 存档预览会自动导入题库吗？

不会。存档预览只显示文件内容，只有在你选择导入后才会写入题库。
