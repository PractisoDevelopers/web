# Managing Questions

Practiso provides the Questions page, Clusters page, question detail pages, and
category detail pages for browsing, editing, categorizing, sharing, and
importing questions.

## Browse Questions

The Questions page lists all questions in your question bank. From there, you
can open question details, preview questions, delete them, or share them.

1. Open the Questions page.
2. Scroll through all questions, or use the search box to filter content.
3. Tap a question to open the question detail page.
4. Touch and hold a question to preview its content.
5. Swipe left on a question to delete it.

## Edit Questions

The question detail page supports both browsing and editing modes.

1. On the Questions page, tap the question you want to modify.
2. On the question detail page, tap the edit button.
3. Modify the question prompt or answer options.
4. Exit editing mode when you are done.

You can also rename a question, share it, or view its categories from the
question detail page.

## Share Questions to the Community

The Questions page lets you share questions to the community.

1. On the Questions page, select the question you want to share.
2. Open the share menu.
3. Choose to share it to the community.
4. Follow the prompts to complete the upload.

Content uploaded to the community is publicly visible. Before sharing, make sure
the question content can be made public.

## Browse Categories

The Clusters page lists all categories. Categories help you organize questions
by topic, chapter, or knowledge point.

1. Open the Clusters page.
2. Tap a category to open its detail page.
3. Touch and hold a category to preview its content.
4. Swipe left on a category to delete it.

## Manage Category Content

The category detail page shows questions that belong to the current category.
You can view questions, adjust relevance, or exclude questions from the
category.

1. Go to the Clusters page.
2. Tap the category you want to manage.
3. On the category detail page, review the questions in that category.
4. Touch and hold a question to preview its content.
5. Open the question actions menu to change relevance or exclude it from the
   category.

## Use Semantic Search to Supplement Categories

The lasso button on the category detail page searches for related questions
based on the category name. It is useful for quickly filling out questions under
a topic.

1. Open a category detail page.
2. Tap the lasso button.
3. Review the semantic search results.
4. Add related questions to the current category.

If the search results are not what you expected, adjust the category name and
try again.

## Preview and Import Archives

When opening a Practiso archive from the system file manager, you can preview
its content before importing it into the database.

1. Open a Practiso archive file in the file manager.
2. Enter the archive preview page.
3. Swipe up or down to switch preview questions.
4. After confirming the content, choose to import the archive into the question
   bank.
5. After import finishes, return to the question sets page or continue browsing.

## FAQ

### Does deleting a question affect existing question sets?

Before deleting, make sure the question is no longer needed. If the question has
been used in question sets or categories, related content may no longer be
complete after deletion.

### What is the difference between categories and question sets?

Categories are for long-term organization of your question bank. Question sets
are for organizing one or more practice sessions. You can organize questions
with categories first, then create question sets from those questions.

### Does archive preview automatically import into the question bank?

No. Archive preview only shows the file content. It is written to the question
bank only after you choose to import it.
