# Creating Question Sets

Question sets organize a group of questions into reusable practice content. You can choose questions manually, accept Practiso's recommendations, or combine both methods.

## Start From Recent Practice

The question sets page shows recently used practice cards. Each card corresponds to one answer record, with colors assigned automatically to help you tell them apart quickly.

1. Open Practiso.
2. Go to the question sets page.
3. Tap a recent practice card.
4. After entering the answering page, swipe up or down to switch questions.

If you want to review the history under a question set first, tap the question set in the list, then choose the corresponding answer record to start.

## Create a Question Set

1. Tap the create button in the upper-right corner of the question sets page.
2. Configure the question set in the new question set panel.
3. Select the questions to add, or use recommendations to supplement the set.
4. After confirming, return to the question sets page. The new question set appears in the list.

## Select Questions Manually

Manual selection works well when you already know the scope of practice, such as reviewing previous mistakes, a chapter, or a custom group of questions.

1. Open question selection from the new question set panel.
2. Select the questions to add to the question set.
3. Return to the new question set panel to confirm your selection.
4. Create the question set.

## Use Question Recommendations

Question recommendations are useful when you want to quickly fill out practice content or expand related practice around existing questions.

1. Review the recommended questions in the new question set panel.
2. Preview the recommendations and confirm whether they match your practice goal.
3. Accept the recommended questions you need.
4. Create the question set, or continue adding other questions manually.

## Start Answering

1. On the question sets page, tap the question set or recent practice card you want to practice.
2. After entering the answering page, swipe up or down to switch questions.
3. If a timer is set, the remaining time appears at the bottom of the screen.
4. If accuracy display is enabled, wrong answers trigger animation and vibration feedback.

## FAQ

### Why are there two entry points on the question sets page?

Recent practice cards let you quickly continue or restart recent answering. The question set list lets you open a question set first, then choose a specific answer record.

### Can I both select questions manually and use recommendations?

Yes. You can manually select the core questions first, then accept recommended questions to expand the practice scope.

### Where do I set the timer?

When creating a question set, choose to start practice at the same time, then use the timer wheel to select the duration. You can also open an existing question set detail page and add a timer the same way when creating a new practice. After setup, the timer appears at the bottom of the answering page.
