# 建立題集

題集用於把一組試題組織成一次可重複練習的內容。你可以手動選擇試題，也可以採納 Practiso 的試題推薦，或將兩種方式結合使用。

## 從最近練習開始作答

題集頁會顯示最近使用的練習卡片。每張卡片對應一次作答記錄，顏色由作答記錄自動分配，方便快速區分。

1. 打開 Practiso。
2. 進入題集頁。
3. 點按最近練習卡片。
4. 進入作答頁後，上下滑動切換題目。

如果你想先查看某個題集下的歷史記錄，可以先點按題集列表中的題集，再選擇對應作答記錄開始。

## 新增題集

1. 在題集頁點按右上角的建立按鈕。
2. 在新增題集面板中設定題集內容。
3. 選擇要加入題集的試題，或使用推薦結果補充題目。
4. 確認後返回題集頁，新的題集會出現在列表中。

## 手動選擇試題

手動選擇適合你已經知道練習範圍的場景，例如複習某一次錯題、某個章節或一組自訂題目。

1. 在新增題集面板中進入試題選擇。
2. 勾選要加入題集的題目。
3. 返回新增題集面板確認選擇結果。
4. 建立題集。

## 使用試題推薦

試題推薦適合你希望快速補齊練習內容，或想圍繞已有題目擴充相關練習的場景。

1. 在新增題集面板中查看推薦試題。
2. 預覽推薦內容，確認是否符合練習目標。
3. 採納需要的推薦題目。
4. 建立題集，或繼續手動加入其他題目。

## 開始作答

1. 在題集頁點按要練習的題集或最近練習卡片。
2. 進入作答頁後，透過上下滑動切換題目。
3. 如果設定了計時器，剩餘時間會顯示在螢幕底部。
4. 如果開啟正確率顯示，答錯時會出現動畫與震動回饋。

## 常見問題

### 為什麼題集頁有兩種入口？

最近練習卡片用於快速繼續或重新開始最近作答；題集列表用於先查看某個題集，再選擇具體作答記錄。

### 可以同時手動選題和使用推薦嗎？

可以。你可以先手動選擇核心題目，再採納推薦題目補充練習範圍。

### 計時器在哪裡設定？

建立題集時選擇同時開始練習，使用計時器轉盤選擇需要的時長。也可進入現有題集詳細頁，在建立新練習時使用相同方式加入計時器。設定完成後，進入作答頁，計時器會顯示在螢幕底部。
