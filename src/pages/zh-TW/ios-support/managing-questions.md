# 管理試題

Practiso 提供題目頁、聚類頁、試題詳細頁和分類詳細頁，用於瀏覽、編輯、分類、分享和匯入試題。

## 瀏覽題目

題目頁列出題庫中的所有題目。你可以從這裡進入試題詳細資訊，也可以對題目執行預覽、刪除和分享操作。

1. 打開題目頁。
2. 捲動瀏覽所有試題，或使用搜尋框篩選內容。
3. 點按題目進入試題詳細頁。
4. 長按題目可預覽內容。
5. 左滑題目可執行刪除操作。

## 編輯試題

試題詳細頁支援瀏覽和編輯兩種狀態。

1. 在題目頁點按需要修改的題目。
2. 在試題詳細頁點按編輯按鈕。
3. 修改題幹或選項內容。
4. 完成編輯後退出編輯狀態。

你也可以在試題詳細頁重新命名試題、分享試題，或查看該試題所屬的分類。

## 分享試題到社群

題目頁支援將題目分享到社群。

1. 在題目頁選擇要分享的題目。
2. 開啟分享選單。
3. 選擇分享到社群。
4. 依提示完成上傳。

上傳到社群的內容會公開展示。分享前請確認題目內容可以公開。

## 瀏覽分類

聚類頁列出所有分類。分類可幫助你依主題、章節或知識點整理試題。

1. 打開聚類頁。
2. 點按分類進入分類詳細頁。
3. 長按分類可預覽內容。
4. 左滑分類可執行刪除操作。

## 管理分類內容

分類詳細頁會顯示屬於目前分類的試題。你可以查看題目、調整相關度，或將題目從分類中排除。

1. 進入聚類頁。
2. 點按需要管理的分類。
3. 在分類詳細頁查看該分類下的試題。
4. 長按題目預覽內容。
5. 開啟題目操作選單，修改相關度或從分類中排除。

## 使用語意搜尋補充分類

分類詳細頁的套索按鈕會根據分類名稱搜尋相關試題，適合快速補齊某個主題下的題目。

1. 進入分類詳細頁。
2. 點按套索按鈕。
3. 查看語意搜尋結果。
4. 將相關題目加入目前分類。

如果搜尋結果不符合預期，可以調整分類名稱後再試。

## 預覽和匯入存檔

從系統檔案管理器打開 Practiso 存檔時，可以先查看內容，不需要立即匯入資料庫。

1. 在檔案管理器中打開 Practiso 存檔檔案。
2. 進入存檔預覽頁。
3. 上下滑動切換預覽題目。
4. 確認內容後選擇將存檔匯入題庫。
5. 匯入完成後返回題集頁或繼續瀏覽。

## 常見問題

### 刪除題目會影響已有題集嗎？

刪除前請確認該題目不再需要。若題目已被用於題集或分類，刪除後相關內容可能不再完整。

### 分類和題集有什麼區別？

分類用於長期整理題庫，題集用於組織一次或多次練習。你可以先用分類整理題目，再基於這些題目建立題集。

### 存檔預覽會自動匯入題庫嗎？

不會。存檔預覽只顯示檔案內容，只有在你選擇匯入後才會寫入題庫。
